# NonIntegerGames

Dependencies
--------------------

* BOOST

Install Boost
    
    sudo apt-get install libboost-dev
    
Compile and run
---------------------

Make source: 

    cd src
    make
    
Pull up the -help options:   
    
    ./game -h
    
Example run:
    
    ./game -f ../benchmarks/eg1.txt -df 2 -p 1 -id 0 -syn
